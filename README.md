# 一天掌握技术基础

AI 时代开发者的必备知识清单，涵盖 23 个核心技术概念。

- 🌐 **在线访问**：https://tech-in-one-day.vercel.app
- 🎨 **设计风格**：日式和纸美学 + 印刷品质感
- 🌍 **双语支持**：中文 / English

## 特性

✨ 23 个核心技术概念  
📚 5 大主题分类（基础知识、AI 助手、开发工具、部署运行、调试优化）  
🌏 中英文双语切换  
📱 响应式设计  
💻 代码示例语法高亮  
📋 内容可复制  

## 技术栈

- 纯静态 HTML/CSS/JavaScript
- Highlight.js 代码高亮
- Google Fonts (Noto Serif SC, Crimson Pro)
